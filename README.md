## 📌 Documento de Visão
O ENADE é um componente curricular obrigatório. Atualmente, os estudantes enfrentam dificuldades no preparo devido à falta de materiais organizados, simulados práticos e feedback sobre desempenho. Este projeto visa especificar um sistema web/mobile de questões comentadas do ENADE para democratizar o acesso a materiais de qualidade.

## 👥 Stakeholders (Público-Alvo)
- Estudantes de graduação (Usuários principais)
- Coordenações de curso (Acompanhamento)
- Professores (Apoio pedagógico)
- Administradores do Sistema (Manutenção)
